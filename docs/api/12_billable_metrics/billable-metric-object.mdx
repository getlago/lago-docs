# The billable metric object

This object represents a billable metric.<br></br>

```json
{
  "billable_metric": {

    "name": "bm_name",
    "code": "bm_code",
    "description": "description",
    "aggregation_type": "sum_agg",
    "field_name": "amount_sum",

  }
```
