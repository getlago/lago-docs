# Organization object

This object represents an organization.<br></br>

## Schema

```json
{
  "organization": {
    "name": "Name1",
    "created_at": "2022-05-02T13:04:09Z",
    "webhook_url": "https://test-example.example",
    "country": "CZ",
    "address_line1": "address1",
    "address_line2": null,
    "state": "state1",
    "zipcode": "10000",
    "email": "org@email.com",
    "city": "city125",
    "legal_name": null,
    "legal_number": null,
    "timezone": "Europe/Paris",
    "email_settings": [
      "invoice.finalized",
      "credit_note.created"
    ],
    "billing_configuration": {
      "invoice_footer": "footer custom",
      "invoice_grace_period": 3,
      "document_locale": "fr",
      "vat_rate": 15.0
    }
  }
}
```

| Attributes | Description |
| -----------| ----------- |
| **name** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Name of the organization |
| **created_at** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> &nbsp &nbsp <br></br><Comment>*ISO 8601 datetime in UTC*</Comment> | Creation date |
| **webhook_url** &nbsp &nbsp <Type>String</Type> | Webhook URL |
| **country** &nbsp &nbsp <Type>String</Type><br></br><Comment>*ISO 3166 (alpha-2)*</Comment> | Country where the organization is located |
| **address_line1** &nbsp &nbsp <Type>String</Type> | Address (first line) |
| **address_line2** &nbsp &nbsp <Type>String</Type> | Address (second line) |
| **state** &nbsp &nbsp <Type>String</Type> | State where the organization is located |
| **zipcode** &nbsp &nbsp <Type>String</Type> | Zip code associated with the organization's address |
| **email** &nbsp &nbsp <Type>String</Type> | Business email address |
| **city** &nbsp &nbsp <Type>String</Type> | City where the organization is located |
| **legal_name** &nbsp &nbsp <Type>String</Type> | Legal name of the organization |
| **legal_number** &nbsp &nbsp <Type>String</Type> | Legal number of the organization |
| **timezone** &nbsp &nbsp <Type>String</Type><br></br><Comment>*TZ database*</Comment> | Timezone of the organization - [learn more](../resources/timezones) |
| **email_settings** &nbsp &nbsp <Type>Array</Type><br></br><Comment>Possible Values are `invoice.finalized` and `credit_note.created`</Comment> | Email settings of the organization |

## Billing Configuration Object

| Attributes | Description |
| -----------| ----------- |
| invoice_footer &nbsp &nbsp <Type>String</Type> | Custom invoice footer |

| document_locale &nbsp &nbsp <Type>String</Type><br></br><Comment>*ISO 639-1*</Comment> | Language code - [learn more](../resources/locales) |
| vat_rate &nbsp &nbsp <Type>Float</Type> | Custom VAT rate |
