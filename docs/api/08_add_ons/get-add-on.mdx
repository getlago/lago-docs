# Get a specific add-on

## Route

```curl title="GET"
/api/v1/add_ons/:code
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"

  curl --location --request GET "$LAGO_URL/api/v1/add_ons/:code" \
    --header "Authorization: Bearer $API_KEY"
  ```
  </TabItem>

  <TabItem value="python" label="Python">

  ```python
  from lago_python_client.client import Client
  from lago_python_client.exceptions import LagoApiError

  client = Client(api_key='__YOUR_API_KEY__')

  client.add_ons.find('code')
  ```
  </TabItem>

  <TabItem value="ruby" label="Ruby">

  ```ruby
  require 'lago-ruby-client'

  client = Lago::Api::Client.new({api_key: '__YOUR_API_KEY__'})

  client.add_ons.get('code')
  ```
  </TabItem>

  <TabItem value="javascript" label="Javascript">

  ```javascript
  await client.addOns.findAddOn("code");
  ```
  </TabItem>

  <TabItem value="go" label="Go">

  ```go
    import "fmt"
    import "github.com/getlago/lago-go-client"

    func main() {
      lagoClient := lago.New().

      addOn, err := lagoClient.AddOn().Get("__YOUR_ADDON_CODE__")
      if err != nil {
        // Error is *lago.Error

      }

      // addOn is *lago.AddOn
      fmt.Println(addOn)
    }
  ```

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The add-on was successfully found

  Returns an [add-on](./add-on-object) object.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "add_on_not_found"
  }
  ```

  The `add-on` was not found.
  </TabItem>
