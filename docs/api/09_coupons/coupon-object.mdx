# Coupon object

This object represents a coupon.<br></br>

## Schema

```json
{
  "coupon": {
    "lago_id": "b7ab2926-1de8-4428-9bcd-779314ac129b",
    "name": "coupon_name",
    "code": "coupon_code",
    "description": "A description of the coupon",
    "amount_cents": 5000,
    "amount_currency": "USD",
    "percentage_rate": null,
    "coupon_type": "fixed_amount",
    "reusable": true,
    "limited_plans": true,
    "plan_codes": ["code1"],
    "limited_billable_metrics": false,
    "billable_metric_codes": [],
    "frequency": "recurring",
    "frequency_duration": 3,
    "expiration": "time_limit",
    "expiration_at": "2022-08-08T23:59:59Z",
    "created_at": "2022-04-29T08:59:51Z"
  }
}
```

| Attributes | Description |
| -----------| ----------- |
| **lago_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifier of the coupon in Lago application. |
| **name** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Coupon name. |
| **code** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Code identifying the coupon. |
| **description** &nbsp &nbsp <Type>String</Type> | Description of the coupon. |
| **amount_cents** &nbsp &nbsp <Type>Integer</Type> | Amount in cents. |
| **amount_currency** &nbsp &nbsp <Type>String</Type> | Currency of the amount. |
| **reusable** &nbsp &nbsp <Type>Boolean</Type> | Field specifying if coupon is reusable or not. |
| **limited_plans** &nbsp &nbsp <Type>Boolean</Type> | Field specifying if coupon has plan limitations or not. |
| **plan_codes** &nbsp &nbsp <Type>Array</Type> | Field specifying related plan codes. |
| **limited_billable_metrics** &nbsp &nbsp <Type>Boolean</Type> | Field specifying if coupon has billable metric limitations or not. |
| **billable_metric_codes** &nbsp &nbsp <Type>Array</Type> | Field specifying related billable metric codes. |
| **expiration** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable><br></br> | Expiration type. <br></br> <details><summary>**Possible values**</summary><div>- `time_limit`<br></br>- `no_expiration`<div></div></div></details> |
| **expiration_at** &nbsp &nbsp <Type>String</Type> <br></br><Comment>*ISO 8601 datetime in UTC*</Comment> | Expiration date and time of the coupon. |
| **coupon_type** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable><br></br> | Coupon type. <br></br> <details><summary>**Possible values**</summary><div>- `fixed_amount`<br></br>- `percentage`<div></div></div></details> |
| **percentage_rate** &nbsp &nbsp <Type>Float</Type> | Percentage rate of the coupon. |
| **frequency** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable><br></br> | Frequency type. <br></br> <details><summary>**Possible values**</summary><div>- `once`<br></br>- `recurring`<div></div></div></details> |
| **frequency_duration** &nbsp &nbsp <Type>Integer</Type> | It determines the number of billing periods to which the coupon applies |
| **created_at** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> &nbsp &nbsp <br></br><Comment>*ISO 8601 datetime in UTC*</Comment> | Date of coupon creation. |
