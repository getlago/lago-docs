# Destroy coupon

## Route

```curl title="DELETE"
/api/v1/customers/:external_id/coupons/:code
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"

  curl --location --request DELETE "$LAGO_URL/api/v1/customers/:external_id/coupons/:code" \
    --header "Authorization: Bearer $API_KEY"
  ```
  </TabItem>

  <TabItem value="python" label="Python">

  ```python
  from lago_python_client import Client

  client = Client(api_key='__YOUR_API_KEY__')

  client.applied_coupons().destroy('external_customer_id', 'coupon_code')
  ```
  </TabItem>

  <TabItem value="ruby" label="Ruby">

  ```ruby
  require 'lago-ruby-client'

  client = Lago::Api::Client.new({api_key: '__YOUR_API_KEY__'})

  client.applied_coupons.destroy('external_customer_id', 'coupon_code')
  ```
  </TabItem>

  <TabItem value="javascript" label="Node.js">

  ```javascript
  let client = new Client('__API_KEY__')

  await client.destroyAppliedCoupon('external_customer_id', 'coupon_code');
  ```
  </TabItem>

  <TabItem value="go" label="Go">

  ```go
    import "fmt"
    import "github.com/getlago/lago-go-client"

    func main() {
      lagoClient := lago.New().
        SetApiKey("__YOUR_API_KEY__")

      appliedCoupon, err := lagoClient.Coupon().AppliedCouponDelete('external_customer_id', 'coupon_code')
      if err != nil {
        // Error is *lago.Error
        panic(err)
      }

      // appliedCoupon is *lago.AppliedCoupon
      fmt.Println(appliedCoupon)
    }
  ```

  </TabItem>

  <TabItem value="csharp" label="C#">

  ```csharp
using System.Collections.Generic;
using System.Diagnostics;
using Org.OpenAPITools.Api;
using Org.OpenAPITools.Client;
using Org.OpenAPITools.Model;

namespace Example
{
    public class DestroyCouponExample
    {
        public static void Main()
        {
            Configuration.Default.BasePath = "https://api.getlago.com/api/v1";
            // Configure HTTP bearer authorization: bearerAuth
            Configuration.Default.AccessToken = "YOUR_ACCESS_TOKEN";

            var apiInstance = new AppliedCouponsApi(Configuration.Default);
            var externalCustomerId = "external_customer_id"; // string | External id of the customer
            var couponCode = "coupon_code"; // string | Code of the coupon

            try
            {
                // Delete an applied coupon
                AppliedCoupon result = apiInstance.DestroyAppliedCoupon(externalCustomerId, couponCode);
                Debug.WriteLine(result);
            }
            catch (ApiException e)
            {
                Debug.Print("Exception when calling AppliedCouponsApi.DestroyAppliedCoupon: " + e.Message );
                Debug.Print("Status Code: "+ e.ErrorCode);
                Debug.Print(e.StackTrace);
            }
        }
    }
}
  ```

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The applied coupon was successfully destroyed

  Returns an [applied coupon](./apply-coupon#the-applied-coupon-object) object.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "customer_not_found"
  }
  ```

  The `customer` was not found.

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "coupon_not_found"
  }
  ```

  The `applied_coupon` was not found.

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "applied_coupon_not_found"
  }
  ```

  The `applied_coupon` was not found.
  </TabItem>
