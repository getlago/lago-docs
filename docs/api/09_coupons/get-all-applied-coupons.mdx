# Get all applied coupons

## Route

```curl title="GET"
/api/v1/applied_coupons?per_page=2&page=1
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash

  API_KEY="__YOUR_API_KEY__"

  curl --location --request GET "$LAGO_URL/api/v1/applied_coupons?per_page=2&page=1" \
    --header "Authorization: Bearer $API_KEY"
  ```

## Arguments

| Attributes | Type | Description |
|--|--|--|
| per_page | String &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | Number of records per page. |
| page | String &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | Page number |
| status | String &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | can be either `active` or `terminated` |
| external_customer_id | String &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | Customer external id |

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  Applied coupons were successfully found

  Returns a list of [applied coupon](./apply-coupon#the-applied-coupon-object) objects.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired
