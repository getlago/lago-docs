# Apply a tax rate to a customer

## Route

```curl title="POST"
/api/v1/customers/:external_customer_id/applied_tax_rates
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"
  EXTERNAL_CUSTOMER_ID="1a901a90-1a90-1a90-1a90-1a901a901a90"

  curl --location --request POST "$LAGO_URL/api/v1/customers/$EXTERNAL_CUSTOMER_ID/applied_tax_rates" \
    --header "Authorization: Bearer $API_KEY" \
    --header 'Content-Type: application/json' \
    --data-raw '{
      "applied_tax_rate": {

      }
    }'
  ```

## Arguments

```json
{
  "applied_tax_rate": {
    "tax_rate_code": "example_code"
  }
}
```

| Attributes | Type | Description |
|--|--|--|
| tax_rate_code | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | Code identifying the tax rate.<br/> It must match the `code` property of one of the organization's tax rates. |

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The coupon was assigned to the customer.

  Returns an [applied tax rate](#the-applied-tax-rate-object) object.
  </TabItem>

  <TabItem value="400" label="HTTP 400">

  ```json
  {
    "status": 400,
    "error": "Bad Request"
  }
  ```

  The `applied_coupon` json root is not present in the request body.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "customer_not_found"
  }
  ```

  **Possible error codes:**
  * `customer_not_found`: The external customer id does not match an existing customer.
  * `tax_rate_not_found`: The tax rate code does not match an existing tax rate.


  </TabItem>
</Tabs>

## The applied tax rate object

This object represents a tax rate assigned to a customer of your business.

## Schema

```json
{
  "applied_tax_rate": {
    "lago_id": "1a901a90-1a90-1a90-1a90-1a901a901a90",
    "lago_customer_id": "1a901a90-1a90-1a90-1a90-1a901a901a90",
    "lago_tax_rate_id": "1a901a90-1a90-1a90-1a90-1a901a901a90",
    "tax_rate_code": "example_code",
    "external_customer_id": "1234",
    "created_at": "2022-04-29T08:59:51Z",
  }
}
```

| Attributes | Description |
| -----------| ----------- |
| **lago_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifer of the applied tax rate in Lago application. |
| **lago_customer_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifer of the customer in Lago application. |
| **lago_tax_rate_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifer of the tax rate in Lago application. |
| **tax_rate_code** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Code identifying the tax rate. |
| **external_customer_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifer of the customer in your application. |
| **created_at** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> &nbsp &nbsp <br></br><Comment>*ISO 8601 datetime in UTC*</Comment> | Date of assignation of the tax rate to the customer. |
