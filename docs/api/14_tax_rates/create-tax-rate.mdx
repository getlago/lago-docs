# Create a tax rate

## Route

```curl title="POST"
/api/v1/tax_rates
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash

  API_KEY="__YOUR_API_KEY__"

  curl --location --request POST "$LAGO_URL/api/v1/tax_rates" \
    --header "Authorization: Bearer $API_KEY" \
    --header 'Content-Type: application/json' \

      "tax_rate": {
  ```

## Arguments

```json
{
  "tax_rate": {
    "name": "tax rate name",
    "code": "tax_rate_code",
    "value": 10.0,
    "description": "tax rate description"
  }
}
```

| Attributes | Type | Description |
|--|--|--|
| name | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | Tax rate name |
| code | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | Code identifying the tax rate |
| value | Float &nbsp &nbsp &nbsp<Required>**Required**</Required> | Value in decimal applied to the tax rate |
| description | String &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | Tax rate description. |
## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The tax rate was created

  Returns a [tax rate](./tax-rate-object) object.
  </TabItem>

  <TabItem value="400" label="HTTP 400">

  ```json
  {
    "status": 400,
    "error": "Bad Request"
  }
  ```

  The `tax rate` json root is not present in the request body.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="422" label="HTTP 422">

  ```json
  {
    "status": 422,
    "error": "Unprocessable entity",
    "code": "validation_errors",
    "error_details": {
      "field": ["error"]
    }
  }
  ```

  **Possible errors:**

  | field | error | description |
  |--|--|--|
  | `name` | `value_is_mandatory` | `name` is missing |
  | `value` | `value_is_mandatory` | `value` is missing |
  | `code` | `value_already_exists` | `code` value is already used for another tax rate |


  </TabItem>
