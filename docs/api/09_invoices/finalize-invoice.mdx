# Finalize a draft invoice

## Route

```curl title="PUT"
/api/v1/invoices/:lago_id/finalize
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"

  curl --location --request PUT "$LAGO_URL/api/v1/invoices/5eb02857-a71e-4ea2-bcf9-57d3a41bc6ba/finalize" \
    --header "Authorization: Bearer $API_KEY" \
    --header 'Content-Type: application/json' \
    --data-raw '{}'
  ```
  </TabItem>

  <TabItem value="python" label="Python">

  ```python
  from lago_python_client import Client

  client = Client(api_key='__YOUR_API_KEY__')

  client.invoices().finalize('5eb02857-a71e-4ea2-bcf9-57d8885990ba') # Invoice ID
  ```
  </TabItem>

  <TabItem value="ruby" label="Ruby">

  ```ruby
  require 'lago-ruby-client'

  client = Lago::Api::Client.new({api_key: '__YOUR_API_KEY__'})

  client.invoices.finalize('5eb02857-a71e-4ea2-bcf9-57d8885990ba') // Invoice ID
  ```
  </TabItem>

  <TabItem value="javascript" label="Node.js">

  ```javascript
  import Client from 'lago-nodejs-client'

  let client = new Client('__API_KEY__')

  await client.finalizeInvoice('id');
  ```
  </TabItem>

  <TabItem value="go" label="Go">

  ```go
    import "fmt"
    import "github.com/getlago/lago-go-client"

    func main() {
      lagoClient := lago.New().
        SetApiKey("__YOUR_API_KEY__")

      invoice, err := lagoClient.Invoice().Finalize("__YOUR_INVOICE_ID__")
      if err != nil {
        // Error is *lago.Error
        panic(err)
      }

      // invoice is *lago.Invoice
      fmt.Println(invoice)
    }
  ```

  </TabItem>
</Tabs>

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>

  The draft invoice has been successfully finalized.

  Returns an [invoice](./invoice-object) object.

  </TabItem>

  <TabItem value="400" label="HTTP 400">

  ```json
  {
    "status": 400,
    "error": "Bad Request"
  }
  ```

  The `invoice` json root is not present in the request body.

  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unauthorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "invoice_not_found"
  }
  ```

  The `invoice` was not found or is already finalized.

  </TabItem>
</Tabs>
