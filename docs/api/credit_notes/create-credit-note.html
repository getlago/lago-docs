import "fmt"
import "github.com/getlago/lago-go-client"

func main() {
  lagoClient := lago.New().SetApiKey("__YOUR_API_KEY__")

  creditNoteInput := &lago.CreditNoteInput{
    LagoInvoiceID: "__LAGO_INVOICE_ID__",
    Reason:        lago.CreditNoteReasonDuplicatedCharge,
    CreditAmountCents 10,
    RefundAmountCents 5,
    Items:         []lago.CreditNoteItemInput{
      {
        LagoFeeID         "__LAGO_FEE_ID__",
        AmountCents 10,
      },
      {
        LagoFeeID         "__LAGO_FEE_ID__",
        AmountCents 5,
      },
    },
  }

  creditNote, err := lagoClient.CreditNote().Create(creditNoteInput)
  if err != nil {
    // Error is *lago.Error
    panic(err)
  }

  fmt.Println(creditNote)
}

Credit note attributes​

AttributesTypeDescription
invoice_idString      RequiredInvoice id in lago application
reasonString      RequiredReason of the credit note creation.
It can be duplicated_charge, product_unsatisfactory, order_change, order_cancellation, fraudulent_charge or other
itemsArray      RequiredList of items. See bellow for items attributes.
credit_amount_centsInteger      RequiredAmount to be credited on customer balance.
refund_amount_centsInteger      RequiredAmount to be refunded to customer.

Credit note item attributes​

AttributesTypeDescription
fee_idString      RequiredFee id in lago application
amount_centsInteger      RequiredAmount to add to the credit note.

Returns a credit note object.

{
  "status": 400,
  "error": "Bad Request"
}

The credit_note json root is not present in the request body.

{
  "status": 401,
  "error": "Unauthorized"
}

Access to the API endpoint is unhautorized.

Possible reasons are:

  • The Authorization header is missing
  • The Authorization header does not contain the API key
  • The Api key is invalid or expired
{
  "status": 404,
  "error": "Not Found",
  "code": "invoice_not_found"
}

Possible error codes:

  • invoice_not_found: The invoice id does not match an existing invoice.
  • fee_not_found: One of the items fee id does not match an existing fee or is not attached to the invoice.
{
  "status": 422,
  "error": "Unprocessable entity",
  "code": "validation_errors",
  "error_details": {
    "field": ["error"]
  }
}

Possible errors:

  • On credit note attributes:

    FieldCodeDescription
    basedoes_not_match_item_amountsItems amount does not match credited and refunded amount
    basehigher_than_remaining_invoice_amountSum of credit and refund amount is higher than the remaining amount of the invoice
    credit_amount_centshigher_than_remaining_invoice_amountCredit amount is higher than the remaining creditable amount of the invoice
    reasonvalue_is_invalidProvided reason does not match an accepted value
    refund_amount_centshigher_than_remaining_invoice_amountRefund amount is higher than the remaining refundable amount of the invoice

  • On credit note items attributes:

    FieldCodeDescription
    amount_centsinvalid_valueAmount is not a positive integer
    amount_centshigher_than_remaining_fee_amountAmount is higher than the remaining amount of the fee
    amount_centshigher_than_remaining_invoice_amountAmount is higher than the remaining amount of the invoice
