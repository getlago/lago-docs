# Destroy applied tax

## Route

```curl title="DELETE"
/api/v1/customers/:external_customer_id/applied_taxes/:tax_code
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash

  API_KEY="__YOUR_API_KEY__"

  curl --location --request DELETE "$LAGO_URL/api/v1/customers/:external_customer_id/applied_taxes/:tax_code" \
  ```

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The applied tax was successfully destroyed

  Returns an [applied tax](../taxes/apply_tax#the-applied-tax-object) object.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "customer_not_found"
  }
  ```

  **Possible error codes:**
  * `customer_not_found`: The external customer id does not match an existing customer.
  * `tax_not_found`: The tax code does not match an existing tax.
  * `applied_tax`: The tax was not applied to the customer.


  </TabItem>
