# Tax object

This object represents a tax.<br></br>

## Schema

```json
{
  "tax": {
    "lago_id": "b7ab2926-1de8-4428-9bcd-779314ac129b",
    "name": "tax name",
    "code": "tax_code",
    "rate": 15.00,
    "description": "tax description",
    "applied_to_organization": false,
    "customers_count": 12,
    "created_at": "2022-04-29T08:59:51Z"
  }
}
```

| Attributes | Description |
| -----------| ----------- |
| **lago_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifier of the tax in Lago application. |
| **name** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Tax name. |
| **code** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Code identifying the tax. |
| **rate** &nbsp &nbsp <Type>Float</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Value in decimal applied to the tax. |
| **applied_to_organization** &nbsp &nbsp <Type>Boolean</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Set to true to use the tax as a default one for customers. |
| **description** &nbsp &nbsp <Type>String</Type> | Tax description. |
| **customers_count** &nbsp &nbsp <Type>Integer</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Number of customers using this tax. |
| **created_at** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> &nbsp &nbsp <br></br><Comment>*ISO 8601 datetime in UTC*</Comment> | Date of tax creation. |
