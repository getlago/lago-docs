# Update a tax

## Route

```curl title="PUT"
/api/v1/taxes/:code
```

## Usage

## Arguments

```json
{
  "tax": {
    "name": "tax name",
    "code": "tax_code",
    "rate": 10.0,
    "description": "tax description",
    "applied_to_organization": false
  }
}
```

| Attributes | Type | Description |
|--|--|--|
| name | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | Tax name |
| code | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | Code identifying the tax |
| rate | Float &nbsp &nbsp &nbsp<Required>**Required**</Required> | Value in decimal applied to the tax |
| description | String &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | Tax description. |
| applied_to_organization | Boolean &nbsp &nbsp &nbsp<Optional>**Optional**</Optional> | Set to true to use the tax as a default one for customers. |

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The tax was successfully updated

  Returns a [tax](./tax-object) object.
  </TabItem>

  <TabItem value="400" label="HTTP 400">

  ```json
  {
    "status": 400,
    "error": "Bad Request"
  }
  ```

  The `tax` json root is not present in the request body.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "tax_not_found"
  }
  ```

  The `tax` was not found.
  </TabItem>

  <TabItem value="422" label="HTTP 422">

  ```json
  {
    "status": 422,
    "error": "Unprocessable entity",
    "code": "validation_errors",
    "error_details": {
      "field": ["error"]
    }
  }
  ```

  **Possible errors:**

  | field | error | description |
  |--|--|--|
  | `name` | `value_is_mandatory` | `name` is missing |
  | `rate` | `value_is_mandatory` | `rate` is missing |
  | `code` | `value_already_exists` | `code` value is already used for another tax |


  </TabItem>
