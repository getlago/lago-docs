# The Add-on object

This object represents an add-on.<br></br>

## Schema

```json
{
  "add_on": {
    "lago_id": "b7ab2926-1de8-4428-9bcd-779314ac129b",
    "name": "add_on_name",
    "code": "add_on_code",
    "amount_cents": 5000,
    "amount_currency": "USD",
    "description": "description",
    "created_at": "2022-04-29T08:59:51Z"
  }
}
```

| Attributes | Description |
| -----------| ----------- |
| **lago_id** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Unique identifier of the add-on in Lago application. |
| **name** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Add-on name. |
| **code** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Code identifying the add-on. |
| **amount_cents** &nbsp &nbsp <Type>Integer</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Amount in cents. |
| **amount_currency** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> | Currency of the amount. |
| **description** &nbsp &nbsp <Type>String</Type> | Description. |
| **created_at** &nbsp &nbsp <Type>String</Type> &nbsp &nbsp <NotNullable>Not null</NotNullable> &nbsp &nbsp <br></br><Comment>*ISO 8601 datetime in UTC*</Comment> | Date of add-on creation. |
