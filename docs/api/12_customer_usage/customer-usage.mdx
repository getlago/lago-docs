# Get customer current usage

## Route

```curl title="GET"
/api/v1/customers/:customer_id/current_usage?subscription_id=id
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"
  CUSTOMER_ID="__CUSTOMER_ID__"
  SUBSCRIPTION_ID="__SUBSCRIPTION_ID__"

  curl --location --request GET "$LAGO_URL/api/v1/customers/$CUSTOMER_ID/current_usage?subscription_id=$SUBSCRIPTION_ID"
    --header "Authorization: Bearer $API_KEY" \
    --header 'Content-Type: application/json'
  ```

  </TabItem>

  <TabItem value="python" label="Python">

  ```python
  from lago_python_client import Client

  client = Client(api_key='__YOUR_API_KEY__')

  customer_usage = client.customers().current_usage('customer_id', 'subscription_id')
  ```

  </TabItem>

  <TabItem value="javascript" label="Node.js">

  ```javascript
  import Client from 'lago-nodejs-client'

  let client = new Client('__API_KEY__')

  let customerUsage = await client.customerCurrentUsage('customer_id', 'subscription_id')
  ```

  </TabItem>

  <TabItem value="ruby" label="Ruby">

  ```ruby
  require 'lago-ruby-client'

  client = Lago::Api::Client.new({api_key: '__YOUR_API_KEY__'})
  customer_usage = client.customer.current_usage('customer_id', 'subscription_id')
  ```

  </TabItem>

  <TabItem value="go" label="Go">

  ```go
    import "fmt"
    import "github.com/getlago/lago-go-client"

    func main() {
      lagoClient := lago.New().
        SetApiKey("__YOUR_API_KEY__")

      customerUsage, err := lagoClient.Customer().CurrentUsage("__YOUR_CUSTOMER_ID__")
      if err != nil {
        // Error is *lago.Error
        panic(err)
      }
        
      // customerUsage is *lago.CustomerUsage
      fmt.Println(customerUsage)
    }
  ```

  </TabItem>
</Tabs>

## Arguments

| Attributes | Type | Description |
| ---------- | ---- | ----------- |
| customer_id | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | Customer unique identifier in your application |

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>

  Returns a [customer usage](./customer-usage-object) object.

  </TabItem>
    <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>
  <TabItem value="404" label="HTTP 404" default>

  ```json
  {
    "status": 404,
    "error": "Not Found"
  }
  ```

  The provided `customer_id` does not match an existing customer.

  </TabItem>
  <TabItem value="422" label="HTTP 422">


  ```json
  {
    "status": 422,
    "error": "Unprocessable entity",
    "message": "message",
    "error_details": {
      "field": ["message"]
    }
  }
  ```

  Possible error messages:
  - `no_active_subscription`: Customer does not have an active subscription


  </TabItem>
</Tabs>
