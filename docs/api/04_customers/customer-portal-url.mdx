# Get customer portal URL

## Route

```curl title="GET"
/api/v1/customers/:external_id/portal_url
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"

  curl --location --request GET "$LAGO_URL/api/v1/customers/:external_id/portal_url" \
    --header "Authorization: Bearer $API_KEY"
  ```
  </TabItem>

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  The customer portal URL was successfully generated

  ```json
  {
    "customer": {
      "portal_url": "https://app.lago.com/customer-portal/abc"
    }
  }
  ```
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>

  <TabItem value="403" label="HTTP 403">

  ```json
  {
    "status": 403,
    "error": "Forbidden",
    "code": "feature_unavailable"
  }
  ```

  The licence needs to be premium in order to use this feature.
  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found",
    "code": "customer_not_found"
  }
  ```

  The `customer` was not found.
  </TabItem>
