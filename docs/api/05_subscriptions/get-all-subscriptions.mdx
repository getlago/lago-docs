# Get all subscriptions for a customer

## Route

```curl title="GET"
/api/v1/subscriptions?customer_id=id
```

## Usage

<Tabs groupId="prog-language">
  <TabItem value="curl" label="Curl" default>

  ```bash
  LAGO_URL="https://api.getlago.com"
  API_KEY="__YOUR_API_KEY__"

  curl --location --request GET "$LAGO_URL/api/v1/subscriptions?customer_id=123" \
    --header "Authorization: Bearer $API_KEY"
  ```
  </TabItem>

  <TabItem value="python" label="Python">

  ```python
  from lago_python_client import Client

  client = Client(api_key='__YOUR_API_KEY__')

  client.subscriptions().find_all({'customer_id': '123'})
  ```
  </TabItem>

  <TabItem value="ruby" label="Ruby">

  ```ruby
  require 'lago-ruby-client'

  client = Lago::Api::Client.new({api_key: '__YOUR_API_KEY__'})

  client.subscriptions.get_all({ customer_id: '123'})
  ```
  </TabItem>

  <TabItem value="javascript" label="Node.js">

  ```javascript
  let client = new Client('__API_KEY__')

  await client.findAllSubscriptions({customer_id: '123'});
  ```
  </TabItem>
</Tabs>

## Arguments

| Attributes | Type | Description |
|--|--|--|
| customer_id | String &nbsp &nbsp &nbsp<Required>**Required**</Required> | External customer id. |

## Responses

<Tabs>
  <TabItem value="200" label="HTTP 200" default>
  Subscriptions were successfully found
  </TabItem>

  <TabItem value="404" label="HTTP 404">

  ```json
  {
    "status": 404,
    "error": "Not Found"
  }
  ```
  The `customer` was not found by external id.
  </TabItem>

  <TabItem value="401" label="HTTP 401">

  ```json
  {
    "status": 401,
    "error": "Unauthorized"
  }
  ```

  Access to the API endpoint is unhautorized.

  **Possible reasons are:**
  * The `Authorization` header is missing
  * The `Authorization` header does not contain the API key
  * The Api key is invalid or expired


  </TabItem>


</Tabs>
